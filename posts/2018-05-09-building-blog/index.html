2.3 软件安装

1) Git软件的安装

  • Ubuntu系统 
sudo apt-get install git
  • Centos系统
sudo yum install git

2）Hugo软件的安装

  • 在Hugo官网上下载 .tar.gz 包
  • 解压可获得二进制包
tar -zxvf hugo_0.54.0_Linux-64bit.tar.gz
  • 将解压出的二进制包 hugo 移动到/bin文件夹下
sudo mv hugo /bin
